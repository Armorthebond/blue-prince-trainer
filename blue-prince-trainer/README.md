# Blue Prince Trainer

A simple Lua-based trainer for Blue Prince, providing memory manipulation and game state monitoring.

## Usage

1. Ensure you have Lua 5.1+ and lua-memory installed.
2. Run `luarocks install lua-memory` or use the provided requirements.
3. Execute `lua src/main.lua`.

## Files

- `src/main.lua` – Entry point
- `src/trainer.lua` – Core trainer logic
- `src/memory.lua` – Memory read/write utilities
- `src/ui.lua` – Simple console UI
- `test/test_trainer.lua` – Unit tests
